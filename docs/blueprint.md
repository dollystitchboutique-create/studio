# **App Name**: Dollystitch Hub

## Core Features:

- Secure Admin Login: Email/password login restricted to pre-approved administrator emails, providing access to business management tools.
- Product Catalogue Management: An interface for adding new jewelry products by answering questions (category, spec, color, price, description, photo uploads), automatically generating suggested unique SKUs that can be modified, and manual soft-deleting items.
- Point-of-Sale Checkout: Streamlined checkout process allowing items to be added to a cart via SKU search or scanning, application of per-item or cart-wide discounts, collection of customer names and payment methods (Cash/Paylah/PayNow), and saving completed sales.
- Sales History Log: A chronological view of all past sales records, including a display of the total revenue collected, with filters, pagination, and the ability to manually soft-delete entries.
- Sales Data Export: Functionality to export filtered sales data to a CSV file for easy external analysis and reporting.
- Sales Insights Dashboard: A simple dashboard featuring a bar chart visualization of units sold by product type, with adjustable date filters to track sales performance and trends.

## Style Guidelines:

- Primary brand color: A tone of pink (#FC809F), reflecting artisanal quality and elegance.
- Background color: A cute baby pink (#FFE7E9), providing a clean and inviting canvas.
- Accent color: A darker shade of pink (#E75480), offering a distinct yet analogous contrast for interactive elements and highlights.
- Headline font: 'Playfair' (serif) for an elegant, high-end feel in titles.
- Body text font: 'PT Sans' (sans-serif) for clear readability in data tables.
- Clean and minimalist line icons to ensure intuitive navigation and emphasize functionality without visual clutter.
- Mobile-first, responsive layouts designed for ease of use on small screens, featuring clean spacing and card-based displays for products and sales entries.
- Subtle, deliberate animations for user feedback and smooth transitions between different views to enhance user experience.
